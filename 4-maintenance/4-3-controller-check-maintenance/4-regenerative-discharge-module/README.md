# 4.3.4 회생 방전 모듈

회생 방전 모듈\(RDM: Regenerative Discharge Module\)은 모터 감속 시 발생하는 회생 전력에 의해 강전 부품이 손상되는 것을 방지하는 모듈입니다. 이 모듈은 보드와 회생 방전 저항, 케이스로 구성되어 있습니다.
